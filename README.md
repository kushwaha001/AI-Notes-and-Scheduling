# AI Notes and Scheduling

AI Notes and Scheduling is a personal assistant app that helps you turn documents, notes, and messages into calendar events and searchable notes.

## What it does

- Accepts documents such as letters, PDFs, scanned images, or typed/spoken notes.
- Extracts key details using an AI model running on a private server: subject, date, time, location, participants, reference numbers, deadlines, and other important information.
- Lets you review and correct extracted details before saving.
- Saves the event to your calendar and stores the original document or note for easy reference.

## Why it helps

- Reduces manual data entry from mail, contracts, meeting invites, and other documents.
- Keeps your schedule and notes synchronized.
- Preserves the original source so you can verify details later.

## How to use

1. Upload a document or enter a note.
2. Review the extracted information.
3. Correct anything that is wrong.
4. Confirm to save the event and note.

## Notes

- The app uses a private AI server for processing, not a cloud-based model.
- Original documents remain accessible from the saved entry.
